# もじれん

Apple Pencil などのペン入力を意識した、文字練習向けの Flutter アプリです。

## 主な機能

- 見本モード
- 書き順ごとの注意表示
- ペン練習ボード
- 筆ペン / ボールペン / えんぴつ切り替え
- 学習したい漢字の登録

## データ

- アプリ表示用データ: `assets/kanji/characters.json`
- 取り込みスクリプト: `tool/build_kanji_assets.dart`
- セットアップ説明: `KANJI_DATA_SETUP.md`

## iPhone 提出

- Codemagic 設定: `codemagic.yaml`
- 手順メモ: `CODEMAGIC_SETUP.md`
