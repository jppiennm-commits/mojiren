# App Icon Source

選んだロゴ案のアイコン元データは以下です。

- [app-icon-source.svg](C:\Users\fermata\書き順アプリ\assets\branding\app-icon-source.svg)

元デザインの別名保存元:

- [ren-logo-gyosho-clear.svg](C:\Users\fermata\書き順アプリ\assets\branding\ren-logo-gyosho-clear.svg)

## 使い方

1. `app-icon-source.svg` を `1024x1024` の PNG に書き出す
2. その PNG を iOS の marketing icon として使う
3. 必要なら同じ元画像から各サイズへ縮小する

## iOS AppIcon 置き場所

- [AppIcon.appiconset](C:\Users\fermata\書き順アプリ\ios\Runner\Assets.xcassets\AppIcon.appiconset)

主なファイル名:

- `Icon-App-1024x1024@1x.png`
- `Icon-App-60x60@3x.png`
- `Icon-App-83.5x83.5@2x.png`

## 補足

この環境では SVG から PNG へ自動変換するツールが見当たらなかったため、今回は「正式なアイコン元データ」と「差し替え先」を整えています。
